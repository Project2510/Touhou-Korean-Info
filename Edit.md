# Editor-Info
동방 프로젝트 입문자를 위한 정보 모음

정보량은 많은데 정작 메인 컨텐츠랄게 없어서 해매는 입문자들을 위한 사이트입니다.
입문 장벽을 낮추기 위해

<ol>
    <li>가시적인 정보량 및 선택지의 최소화</li>
    <li>다양한 컨텐츠에 접근성 상승</li>
</ol>

을 목적으로 하고 있습니다.

이미 발을 담그기 시작한 입문자들에게는 많은 걸 자세히 알려주기보다 무작위 정보에 접근시키는게 입문 장벽을 낮추는데 도움이 될 것이라는 생각입니다.

음악 등의 리스트는 ./csv 폴더에 csv 파일 형태로 태그를 포함해 관리합니다.

<ul>
    <li>캐릭터 : links_char.csv</li>
    <ul>
        <li>네임드 캐릭터들의 나무위키 링크입니다. 운잔 같은 건 제외.</li>
    </ul>
    <li>OST : links_ost.csv</li>
    관리의 편의를 위해 유튜브 일부 공개 링크를 사용합니다. 다만 저작권 자동 검열로 삭제되는 경우가 주로 수왕원 이후부터 발생하는데, 가능하다면 전체 공개된 공식 음원으로, 없다면 외부 링크를 사용합니다.
    <ul>
        <li>정규작: 플레이어즈 스코어와 신령묘 트랜스 모드 제외 모든 신작 트랙입니다.</li>
        <li>외전작: 슈팅게임류는 어레인지되지 않은 곡들을 제외, 격투게임류는 메인 보스와 관련 없는 대화곡, 배경곡, 어레인지곡 등을 제외했습니다. 필요에 의한 일부 예외는 있습니다.</li>
        <li>서적: 봉래인형과 비봉 앨범의 오리지널 곡들입니다.</li>
    </ul>
    <li>동인지 : links_dj.csv</li>
    <ul>
        <li>Regbu 동방 프로젝트 태그 랜덤 링크입니다.</li>
    </ul>
    <li>논보컬 어레인지 : links_nonvc.csv</li>
    <ul>
        <li>메들리류나 원작풍 어레인지류는 넣기 애매해서 제외했습니다.</li>
    </ul>
    <li>보컬 어레인지 : links_vc.csv</li>
    <ul>
        <li>서클색이 너무 진하다 싶은 작품(아카츠키 레코드, foreground eclipse 등...)이나 전파계류는 최대한 제외했는데, 추가해도 별 상관 없을 것 같기도 합니다.</li>
    </ul>
    <li>명작선 : 만화와 음악은 유입을 이끌었거나 바이럴되었던 작품 위주로 우선 선별했으며, 2차 창작 게임은 평가가 좋으며 추천할 만한 게임을 선별했습니다. 더 적절한 작품이 있다면 추가나 수정 바랍니다. 추가로 소설이나 영상물 등의 리스트도 고려 중입니다.
    <ul>
        <li>만화 : links_mp_dj.csv
        <li>음악 : links_mp_music.csv
        <li>2차 창작 게임 : links_mp_game.csv
    </ul>
    </li>
    <li>이벤트 : events.csv</li>
    <ul>
        <li>해외의 큰 이벤트(예대제, 신작 발표, 인기투표 등)와 국내 이벤트 위주 작성 예정입니다.</li>
        <li>events.csv가 없거나 비어있으면 이벤트 버튼이 뜨지 않습니다. 관리가 어려우면 그냥 없애놓는 것도 가능합니다.</li>
    </ul>
    <li>QnA : faq.csv</li>
    <ul>
        <li>많은 내용을 담기보다 핵심 질문 위주로 간결하게 작성했습니다.</li>
    </ul>
</ul>

웬만해선 깃허브 아이디 알려주시면 collaborator 추가해 드립니다.
csv 파일 리스트 업데이트 같은 사소한 업데이트는 양식과 목적에 맞다면 편하게 하셔도 됩니다.
코드 개선 등도 가능하지만 큰 업데이트가 있을 때는 합의가 되었으면 합니다.

Repository 주인이 엔지니어가 아니라 답답한 부분이 있을 수 있으니 양해 부탁드립니다.
